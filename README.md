# Assessment MIFX Automation Projects

This repository contains automation testing projects for assessment purposes.

## Projects

1. **API - Reqres**: API testing project using Katalon Studio for testing the Reqres API.
2. **Assesment MIFX - Sauce Demo Automation**: Web UI testing project for the Sauce Demo application.

## Prerequisites

- [Katalon Studio](https://www.katalon.com/download/) (version 8.3.0 or later recommended)
- Java JDK (version 8 or later)
- Git (for cloning the repository)

## Setup

1. Clone this repository:
   ```
   git clone <repository-url>
   cd assesment-mifx
   ```

2. For each project, follow the specific setup instructions in their respective README.md files.

## Running Tests

- Open the project in Katalon Studio.
- Execute test suites from the Test Suites folder.
- For CI/CD, use the provided build.gradle or azure-pipelines.yml configurations.
